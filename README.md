# AI Product Portfolio

A clean static personal portfolio website for Heriberto "Eddie" Rodriguez, an AI-enabled senior product leader.

The site is intentionally simple: HTML, CSS, and a tiny bit of vanilla JavaScript. It is ready to run locally and suitable for GitHub Pages.

## Sections

- Header and navigation
- Hero
- Credibility strip
- Featured AI Projects
- Career Impact
- About
- Off the Clock
- How I Work
- Resume
- Contact
- Footer

## Preview locally

Open `index.html` directly in a browser, or run a small local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
